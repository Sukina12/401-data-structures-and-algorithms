# Queue With Stack :

### Challenge : 

   * Implement a Queue using two Stacks.

### Approach and Efficiency :

  * space : O(n)
  * time : O(n)

### API :

   #### Stack :
    * push()
    * pop()
    * peek()
    * size()
    * isEmpty()
   #### Queue :
     * enqueue()
     * dequeue()
     * peek()
     * size()
     * isEmpty()
